# Pulse News Project

PulseNews is a high-quality, editorial-style React news aggregator.

## Features
- Staggered animations with Framer Motion
- Dark theme optimized for readability
- Live search and category filtering
- Bookmark functionality with Firebase
- Responsive design

## Tech Stack
- React 19
- Vite
- Firebase (Auth & Firestore)
- Lucide React Icons

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` file with your NewsAPI and Firebase keys
4. Run locally: `npm run dev`
